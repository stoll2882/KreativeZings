CPSC 314 Final Project Presentation

Kreative Zings Ecommerce Webapp

Sam Toll

I currently run my own small business, Kreative Zings, that sells personalized stickers I designed myself. My project recreated this site from scratch, with specific modifications. I used HTML and CSS coupled with React in order to design a professional looking webapp, and brought Node.js and Express.js to complete the back-end development such as adding an item to cart, checkout process etc. Mongo DB was used for data storage such as customer information, completing my use of the MERN stack for this project!

Functional Requirements:

  1. User can navigate between pages using a navigation bar (frontend)

  2. User can login (frontend)

  3. User can login (backend)

  4. User can create an account with confirmation message (frontend)

  5. User can create an account (backend)

  6. User can add an item to their cart (frontend w/ backend component)

  7. User can remove an item from their cart (frontend w/ backend component)

  8. User can upload their own image (frontend)

  9. User can upload their own image (backend)

  10. User can sibmit custom order form (email - backend)

  11. User can submit contact me form if they have questions (with email - backend)

  12. ***Data sanitation and validation (overall - backend)
